## Draft notes from Notion

- Use sentence case everywhere.
- Avoid repeating context the surrounding chrome already provides. Don't title a button "Update broadcast" inside the broadcast page.
- Empty states:
  - "No broadcasts yet"-style headline.
  - Explain what the feature is and what the user can do next.
  - Provide a CTA.
